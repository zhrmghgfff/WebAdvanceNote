# 关键字

|  关键字 |  描述 |   
|---|---|
|  $ | 变量标识 |
|  global | （函数内访问的）全局变量 |
|  $GLOBALS[index] | 全局变量数值,index全局变量名字 |
|  static |  静态变量 |
|  define("GREETING", "欢迎访问") | 常量GREETING（不区分大小写） |
|  . | 字符串运算符，将两个字符串变量连接在一起|
